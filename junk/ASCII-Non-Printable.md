# ASCII Non-Printable Characters (0–31)

| Decimal | Hex  | Binary    | Abbreviation | Name                  | Description                              |
|---------|------|-----------|--------------|----------------------|------------------------------------------|
| 0       | 0x00 | 0000 0000 | NUL          | Null                  | Null character                           |
| 1       | 0x01 | 0000 0001 | SOH          | Start of Heading      | Start of message heading                 |
| 2       | 0x02 | 0000 0010 | STX          | Start of Text         | Marks the start of text body             |
| 3       | 0x03 | 0000 0011 | ETX          | End of Text           | Marks the end of text body               |
| 4       | 0x04 | 0000 0100 | EOT          | End of Transmission   | Ends a transmission                      |
| 5       | 0x05 | 0000 0101 | ENQ          | Enquiry               | Requests a response from a receiver      |
| 6       | 0x06 | 0000 0110 | ACK          | Acknowledge           | Positive acknowledgment signal           |
| 7       | 0x07 | 0000 0111 | BEL          | Bell                  | Triggers an audible or visual alert      |
| 8       | 0x08 | 0000 1000 | BS           | Backspace             | Moves cursor one position back           |
| 9       | 0x09 | 0000 1001 | HT           | Horizontal Tab        | Moves cursor to next tab stop            |
| 10      | 0x0A | 0000 1010 | LF           | Line Feed             | Moves cursor to next line                |
| 11      | 0x0B | 0000 1011 | VT           | Vertical Tab          | Moves cursor down to next vertical tab   |
| 12      | 0x0C | 0000 1100 | FF           | Form Feed             | Advances to the next page or screen      |
| 13      | 0x0D | 0000 1101 | CR           | Carriage Return       | Moves cursor to beginning of line        |
| 14      | 0x0E | 0000 1110 | SO           | Shift Out             | Switches to alternate character set      |
| 15      | 0x0F | 0000 1111 | SI           | Shift In              | Returns to default character set         |
| 16      | 0x10 | 0001 0000 | DLE          | Data Link Escape      | Escapes following characters             |
| 17      | 0x11 | 0001 0001 | DC1          | Device Control 1      | Device control (often XON for flow ctrl) |
| 18      | 0x12 | 0001 0010 | DC2          | Device Control 2      | Device control (secondary)               |
| 19      | 0x13 | 0001 0011 | DC3          | Device Control 3      | Device control (often XOFF for flow ctrl)|
| 20      | 0x14 | 0001 0100 | DC4          | Device Control 4      | Device control (stop)                    |
| 21      | 0x15 | 0001 0101 | NAK          | Negative Acknowledge  | Negative acknowledgment signal           |
| 22      | 0x16 | 0001 0110 | SYN          | Synchronous Idle      | Keeps synchronization in transmission    |
| 23      | 0x17 | 0001 0111 | ETB          | End of Trans. Block   | Marks end of a transmission block        |
| 24      | 0x18 | 0001 1000 | CAN          | Cancel                | Cancels previous data                    |
| 25      | 0x19 | 0001 1001 | EM           | End of Medium         | Indicates end of storage medium          |
| 26      | 0x1A | 0001 1010 | SUB          | Substitute            | Replaces invalid or unrecognized char    |
| 27      | 0x1B | 0001 1011 | ESC          | Escape                | Starts an escape sequence                |
| 28      | 0x1C | 0001 1100 | FS           | File Separator        | Separates files in data streams          |
| 29      | 0x1D | 0001 1101 | GS           | Group Separator       | Separates groups within a file           |
| 30      | 0x1E | 0001 1110 | RS           | Record Separator      | Separates records within a group         |
| 31      | 0x1F | 0001 1111 | US           | Unit Separator        | Separates units within a record          |
